# How to play the game

This app is a simple multi-player real-time quiz game just like Kahoot. Each player joins the game from their phone and whoever gets the highest point wins.

## Steps

1. Host creates a game by clicking the "Start Game" button.
2. Host shares the QR code with the players. 
3. Players scan the QR code and enter the game with a nickname.
4. Once all the players have joined, the host can start the game.
5. A question will be displayed on the host's screen and the players' device. After 5 seconds, the choices are revealed, and the players can choose their answer by tapping the choice.
6. Once all the question is completed, the results are displayed on the host's screen.

## Tips for winning

The faster you answer, the more points you get. So, be quick!
